# Assets Documentation

## Files to Add

### 1. tekken3-screenshot.jpg
This is the game screenshot that will be displayed in the popup window background.

**Recommended Specifications:**
- Resolution: 800x600 pixels or higher
- Format: JPG or PNG
- Filename: `tekken3-screenshot.jpg`

You can obtain Tekken 3 screenshots from:
- Visit https://tekken3.cc and take a screenshot
- Download official screenshots from game resource websites
- Use any Tekken 3 gameplay screen

### 2. icon.png (Optional)
Extension icon file.

**Recommended Specifications:**
- Resolution: 128x128 pixels
- Format: PNG (supports transparent background)
- Filename: `icon.png`

If not provided, Plasmo will use the default icon.

## Placeholder Images

During development, if images are not ready yet, popup.tsx will display a placeholder background.
